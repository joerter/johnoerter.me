Presented at the June 2016 Omaha Dotnet User Group meeting

ASP.NET Core has unlocked the power of Docker for .NET developers. This talk is a deep dive into developing an ASP.NET application inside a Docker container and moving that application through a continuous delivery workflow.  
Objectives:  
- Increased understanding of Docker
- Understand how Docker fits in a continuous delivery workflow
- Comfortable building your own Docker powered ASP.NET Core application
